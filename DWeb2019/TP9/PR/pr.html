Abstract

Este trabalho consisitia em dois exercícios. Um deles seria construir um servidor que separava a API da Interface web. O outro seria implementar uma versão própria do mongoimport.

Para o primeiro exercício utilizamos um ficheiro .json para ser a base da nossa base de dados em Mongo. Esse de dados está em: compositores.json .

Para o segundo exercício tínhamos de dar como argumento o nome da base de dados, o nome da coleção e um ficheiro .json . Para teste utilizei o ficheiro usado no outro exercício, que também podemos encontrar em compositores.json .

Neste segundo exercício podemos ver como correr o exercício com o comando node mongoimport --h
