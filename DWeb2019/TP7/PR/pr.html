Abstract

Este trabalho consiste na construção de um servidor que suporta CRUD, direcionado a uma base de dados de filmes.

Para tal utilizamos um ficheiro .json para ser a base da nossa base de dados em Mongo. Esse de dados está em: movies.json .

Este ficheiro foi carregado para a base de dados em Mongo.

As páginas HTML são geradas a partir de ficheiros pug.
