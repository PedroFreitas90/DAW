Abstract

Este trabalho consiste na construção de um servidor capaz de responder a pedidos de GET e POST .

Para tal foi construida uma pagina web capaz de submeter dados e de seguida apresentá-los numa tabela.

Estes dados serão armazenados num ficheiro json.

Além disso, o servidor é capaz detetar pedidos que não suporta dando como resposta uma página html de erro,evitando assim o erro 404.

As páginas HTML são geradas a partir de ficheiros pug.
